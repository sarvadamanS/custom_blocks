Modal Window Maker

Modal Window Maker Configuration

This is a simple activity with use of Blockly library's Custom blocks.
Here use can use the blocks to make a Custom Modal window,by giving inputs for text,color & other properties to content.
Also for use can play around by adding the random block which works with color & height-width property.
Note:The Create Modal block is necesarry for modal to be displayed without it the other block wont do their work.

Tap the button to edit codespace.
When complete, press Done.
